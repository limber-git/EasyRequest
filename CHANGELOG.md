# Changelog

Todos los cambios relevantes de EasyRequest se documentan en este archivo.

## 0.1.0

- Editor de colecciones `.erequest` con entornos, variables y secretos locales.
- Cliente HTTP con ráfagas limitadas, cancelación, timeout y truncado seguro.
- Descubrimiento OpenAPI 3, Swagger 2 y análisis estático de ASP.NET Core.
- Protección ante documentos inválidos, conflictos de escritura y cambios externos.
- Webview autocontenido con CSP estricta y sin dependencias de scripts remotos.
