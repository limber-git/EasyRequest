# Soporte

Para reportar un bug o solicitar una mejora, abre un issue en
<https://github.com/limber-git/EasyRequest/issues> e incluye:

- versión de EasyRequest y VS Code;
- sistema operativo;
- pasos mínimos para reproducir el problema;
- mensajes visibles, sin tokens, passwords ni datos privados.

Antes de publicar una colección de ejemplo, elimina secretos escritos
directamente en URLs, headers o bodies.
